# 🍽️ TasteBuds Catering System (TBCS)

## Description
TasteBuds Catering System (TBCS) is a Java console-based application developed to simulate the order management process of a restaurant in Dhaka. The system tracks customers, orders, kitchen preparation, delivery handling, and customer feedback.

The design follows basic Object-Oriented Programming principles and fulfills the given scenario requirements.

---

## Functional Flow
The system demonstrates the following outputs:

1. Placing an Order  
2. Kitchen Preparation (Head Chef)  
3. Delivery Assignment  
4. Driver Checkout  
5. Customer Feedback  

All interactions are displayed through console output.

---

## Key Features
- Registered customers receive discounts based on monthly order count
- Orders are assigned queue numbers automatically
- Head Chef manages order priority and preparation details
- Delivery Manager assigns drivers based on order type
- Drivers verify delivery using license numbers
- Customers confirm delivery and submit ratings and feedback
