# 🧑‍🎓 Student Record Management System

## Overview
This project implements a simple **Student Record Management System** using **Java Generics** and the **Java Collections Framework**. The system stores student information and provides basic operations such as sorting, searching, and grouping student records.

## Features
- Store student records using a **generic storage class**.
- **Sort students by GPA**.
- **Search for a student by Student ID**.
- **Group students by gender** using a `Map`.

## Technologies Used
- **Programming Language:** Java  
- **Concepts:** Java Generics, Collections Framework  
- **Data Structures:** `ArrayList`, `HashMap`
